# Coffee_Shop_Sales

1) Project Title / Headline

☕ Coffee Shop Sales Analysis Dashboard (Power BI Project)

2) Short Description / Purpose

This project focuses on analyzing coffee shop sales data to uncover business insights such as revenue trends, customer behavior, and product performance. The dashboard helps stakeholders make data-driven decisions by visualizing key metrics like total sales, profit, and order patterns.

3) Tech Stack
Power BI – Dashboard creation & data visualization
Excel (.xlsx) – Data source & preprocessing
DAX (Data Analysis Expressions) – Calculations & measures
SQL (optional if used) – Data querying (if applicable)
4) Data Source
Dataset: Coffee Shop Sales Dataset
Format: Excel file (.xlsx)
Contains:
Sales Amount
Product Categories
Order Date
Quantity Sold
Profit / Revenue

(You can also write: “Dataset provided for practice / simulated business dataset” if not from a company)

5) Features & Highlights
📊 Interactive Power BI Dashboard
📅 Monthly & Daily Sales Trends Analysis
🛍️ Top-Selling Products Identification
📍 Region-wise / Category-wise Performance
💰 Profit & Revenue Insights
🔍 KPI Cards (Total Sales, Orders, Profit)
📈 Data-driven decision support visuals
